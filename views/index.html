Game Instructions

Tap the 'Start Game' button on the main screen or the 'New Game' link at the top of the game screen.

A duck is automatically selected. Use the rotation buttons at the bottom of the screen to rotate the duck upright.

Tap the 'Save Me' button at the bottom of the screen to rescue the upright duck from impending explosions.

Each duck saved is worth the same amount of points as the level number.
Ex: Level 3 duck saved is worth 3 points.

Good luck!
